# Code of Conduct

Brigade adopts the **Contributor Covenant, version 2.1** as its code of conduct.

The full text is maintained by the Contributor Covenant project and is available
here:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — issues, pull requests, discussions, or any
other community space — you agree to abide by its terms: be respectful and
inclusive, assume good faith, and help keep the community welcoming and
harassment-free.

## Reporting

If you experience or witness unacceptable behavior, please report it privately to
the maintainers via the repository's **Security → Report a vulnerability** form,
or by opening a confidential channel with a maintainer. Reports will be reviewed
and handled with discretion.

> Maintainers: set a dedicated conduct contact address here (e.g.
> `conduct@brigade-agent.ai`) before public launch.
